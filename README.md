# Web Apps

A lightweight terminal utility for creating desktop web applications on Linux.

## Features

- Install websites as desktop apps
- Automatic Icon download
- Application launcher integration
- Multi-select uninstall
- Uses your existing browser session

## Dependencies

- bash
- gum
- jq
- curl

## Installation

```bash
git clone https://github.com/PamacKR/Web-Apps.git

cd Web-Apps

chmod +x install.sh

./install.sh
```

## Usage

web-apps

or

install-web-apps

or

uninstall-web-apps
